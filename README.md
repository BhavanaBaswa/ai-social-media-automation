# AI Social Media Automation (No-Code)

## Overview
This project is a no-code automation built using Make (Integromat), Google Sheets, and AI.
It automatically summarizes content and posts personalized messages to LinkedIn, Telegram, and Facebook.

## Tools Used
- Make (Integromat)
- Google Sheets
- Google Gemini AI
- LinkedIn API
- Telegram Bot API
- Facebook Pages API

## Workflow
1. Google Sheets acts as the trigger and data source.
2. AI summarizes the content.
3. Router branches the workflow.
4. Platform-specific AI personalizers rewrite the content.
5. Posts are published automatically.
